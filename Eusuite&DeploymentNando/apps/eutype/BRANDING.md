# EUTYPE Branding Guide

## Brand Identity

**Naam**: EUTYPE  
**Tagline**: Professionele Tekstverwerker  
**Kleuren**:
- Primary: #0078d4 (Microsoft Blue)
- Secondary: #2b579a (Dark Blue)
- Accent: #0066b8

## Logo

Het EUTYPE logo bestaat uit:
- **EU** in grote, vette letters (verwijzing naar Europa)
- **TYPE** eronder in middelgrote letters
- Document/papier achtergrond in wit
- Blauwe kleurenschema

## Visuele Elementen

### Kleuren
- **Hoofdkleur**: #0078d4 (gebruikt in statusbalk, logo, highlights)
- **Donker blauw**: #2b579a (bestand menu, accenten)
- **Wit**: Documentachtergrond, schone interface
- **Grijs**: #f3f3f3 (ribbon achtergrond)

### Typografie
- **Primary font**: Segoe UI (Windows native)
- **Document font**: Calibri (default voor documenten)
- **Monospace**: Courier New (code blokken)

### Logo Gebruik
- Linker bovenhoek van de ribbon toolbar
- Wit op blauwe achtergrond (#2b579a)
- Letterafstand: 2px
- Font weight: 700 (bold)
- Font size: 18px

### Statusbalk
- Achtergrond: #0078d4
- EUTYPE branding links in statusbalk
- Witte tekst
- Letter spacing: 2px voor brand naam

## Bestandsformaat

**.ty** - EUTYPE Document
- Native formaat
- JSON-based met metadata
- Type: "EUTYPE Document"
- Versie: 1.0

## Applicatie Titels

- **Desktop**: EUTYPE - Professionele Tekstverwerker
- **Web**: EUTYPE - Professionele Tekstverwerker
- **Dialogen**: Standaard Windows/macOS dialogen met EUTYPE Document filters

## Brand Voice

- **Professioneel**: Zakelijk en betrouwbaar
- **Europees**: Focus op Europese markt
- **Modern**: Clean, moderne interface
- **Toegankelijk**: Nederlands als standaard taal

## File Extensions

- `.ty` - EUTYPE Document (primary)
- `.pdf` - Export formaat
- `.txt`, `.html`, `.rtf` - Ondersteund voor import/export
